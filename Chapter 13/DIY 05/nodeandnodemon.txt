How to Run a Node.js Script

Method 1: Run with Node.js
Open the terminal in VS Code by pressing Ctrl + ~ (Windows/Linux) or Cmd + ~ (Mac).

Run your script using:
on bash
node <your-script-name.js>
Example:
on bash
node index.js

Method 2: Run with Nodemon
In the terminal, start your script with 

Nodemon:
on bash
nodemon <your-script-name.js>
Example:
on bash
nodemon index.js
